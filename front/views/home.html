Pour utiliser le service d'api pour afficher vos cagnottes streamlabs charity sur vos streams. Vous pouvez utilsier ces deux liens :

Pour utiliser la cagnotte globale :
Vous devez copier la fin de l'url de la page globale de l'event streamlabs charity.
Imaginons que votre url soit celle là : https://streamlabscharity.com/la-spa/live-spa-contre-labandon
Votre URL que vous devez mettre sur obs est : http://funixgaming.fr:9777/global/la-spa/live-spa-contre-labandon

Pour utiliser la cagnotte personelle :
Vous devez copier la fin de l'url de la page globale de l'event streamlabs charity.
Imaginons que votre url soit celle là : https://streamlabscharity.com/@funixgaming/funixgaming-spa
L'url que vous devez mettre sur obs est : http://funixgaming.fr:9777/streamer/funixgaming/funixgaming-spa sans le @

Basez vous toujours sur la fin de l'url après le https://streamlabscharity.com/
Utilisez bien /streamer pour la cagnotte perso
Et /global pour la cagnotte globale
